# Контрольные вопросы

1. Какую информацию содержит учётная запись пользователя?

Учётная запись пользователя в Linux содержит следующую информацию:

* Имя пользователя,
* Пароль,
* UID (User ID),
* GID (Group ID),
* Домашний каталог,
* Оболочка (Shell),
* Информация о пользователе.

2. Как удалить зависший процесс?
